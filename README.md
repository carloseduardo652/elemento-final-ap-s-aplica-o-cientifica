o programa foi desenvolvido para descobrir com melhor eficiência do que ocorre com elemento após aplicação cientifica especifica.
este trabalho é um modelo inicial com 3 elementos especificos e 4 conceitos fisicos e suas medidas especificas, mas o modelo pode ser ampliado para incluir inumeras possibilidades 
foi utilizado IA para otimiza o resultado
em geral o objetivo é usar um referencial especifico que ocorreu para entender os demais casos poucos explorados como em viagens espacial. 
o pragrama foi desenvolvido baseado no meu proprio artigo cientifico
